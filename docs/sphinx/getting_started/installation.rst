Installing Naaya
================

.. toctree::

    buildout
    buildout_windows

The recommended way to install Naaya is using `zc.buildout`_. For \*nix systems see :doc:`buildout`.

For Windows, see :doc:`buildout_windows`.

If you prefer manual installation read :doc:`manual_installation`.

For old Naaya versions check the README_ for a quick guide.

.. _zc.buildout: http://pypi.python.org/pypi/zc.buildout
